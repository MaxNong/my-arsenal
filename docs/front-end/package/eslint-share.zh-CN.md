## 基本介绍
利用eslint share config 分类管理工程代码eslint规范，直接引用npm包

## 使用

```sh
yarn add @eslint-share/eslint-config-prettier-react-ts -D
```

项目中使用

```js
module.exports = {
  extends: ["@eslint-share/eslint-config-prettier-react-ts"]
};
```
